
## Dengue Competition 


Welcome to the Dengue Competition repository. This repository contains all the necessary files and instructions for the modeling process of the Dengue competition.

## Structure
 - modeling/: Contains the Jupyter notebook detailing the modeling process.
 - train.py: Script to train the model.
 - inference.py: Script to generate predictions.

## Setup

### Install Dependencies
Before you run any script, make sure all required Python packages are installed. You can install them using:

pip install -r requirements.txt

### Usage
To train the model, use the following command: 
python train.py

### Generating Predictions
Once the model is trained, you can generate predictions using:
python inference.py

### Note
Please ensure that you've installed all modules from the requirements.txt file before executing any script.
